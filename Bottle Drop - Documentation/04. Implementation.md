# Technologies

- **Java + Spring Boot**: Core backend framework for creating RESTful microservices.
- **Docker**: Used to containerize and manage service instances.
- **H2 DataBase**: Database for persisting orders, stock levels, and user roles.
- **UML Tools (Mirro)**: Used for modeling domain and system architecture.

# Solution Architecture

Each microservice handles its own logic and database. Services communicate using REST APIs, with asynchronous simulations for delivery-related tasks.
