# Bottle Drop Platform

A distributed microservice-based platform for managing beverage supply and bottle return logistics at large-scale events such as festivals or conferences. Inspired by the c3bottles system used at Chaos Computer Club events, this platform enables centralized warehouse management, distributed bar setups, and bottle drop points, along with real-time inventory tracking and role-based user access.
